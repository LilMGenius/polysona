# Polysona Mid Deck Design Notes

## Visual Thesis
차갑고 선명한 agent-native 무드 위에 README의 aurora SVG를 메인 시그니처로 쓰고, 큰 타이포와 실전 지표를 전면에 세워 "generic AI"가 아니라 "나를 운영하는 시스템"이라는 에너지를 전달한다.

## Content Plan
- Opener: Polysona 정체성과 이름의 의미
- Support / Proof: 왜 기존 에이전트가 부족한지, 왜 지금 돈과 반응으로 연결되는지
- Detail / Story: 인터뷰 -> 구조화 -> 생성 -> QA 흐름
- Close: 랄프 운영력과 현재 MVP 상태
