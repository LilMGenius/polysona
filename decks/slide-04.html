심리학적 Agent Harness 기반 멀티 페르소나 운영 시스템

04

Polysona는 인터뷰와 메타인지를 통해 사용자를 구조화하고, 그 사람에 맞는 멀티 페르소나 콘텐츠 운영 루프를 만듭니다.

01 Interview

나를 묻는다

  • 10-framework 기반 인터뷰
  • 심리학적 / 철학적 단서 수집
  • 표면 니즈와 내면 패턴 분리

02 Structuring

나를 구조화한다

  • persona.md / nuance.md / accounts.md 생성
  • 상황별 여러 가면을 분리 관리
  • generic prompt 대신 persona harness 적용

03 Generation

가장 "나"다운 콘텐츠를 만든다

  • 플랫폼별 초안 자동 생성
  • 말투, 관점, 역할별 페르소나 반영
  • 어떤 AI agent 위에서도 동작

04 QA Loop

virtual follower로 검증한다

  • 버추얼 팔로워가 반응 시뮬레이션
  • 드래프트 QA 후 상위안 추천
  • 출판과 피드백 루프까지 연결

MVP demo: 인터뷰 -> 메타인지 기반 페르소나화 -> 콘텐츠 생성 -> QA

Promise: "AI를 붙이는 것"이 아니라 "나를 운영 가능한 시스템으로 만드는 것"
